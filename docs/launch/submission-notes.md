# Cashenza custom-bundle Submission Notes

## Positioning for review
Cashenza custom-bundle helps merchants create configurable product bundles directly on product pages with per-item variant selection, best seller merchandising, timer urgency, and Shopify-native discount behavior.

## What makes the app different
- Customers can choose variants per article inside the selected bundle
- Merchants can visually highlight the best seller offer
- Merchants can style the block without code
- The app focuses on a cleaner and faster bundle setup than heavyweight alternatives

## Reviewer test flow
1. Open the embedded app
2. Create or edit an active bundle
3. Add the theme app block on the correct product template
4. Open the product page
5. Select a bundle offer
6. Choose variants per item if enabled
7. Add the bundle to cart
8. Verify the discount is reflected in cart behavior

## Reviewer talking points
- The app is embedded in Shopify Admin
- Merchants configure bundles without editing code
- Theme app extension renders the bundle block on product pages
- Shopify-native discount logic powers the cart discount behavior
- Billing is handled through the Starter subscription flow

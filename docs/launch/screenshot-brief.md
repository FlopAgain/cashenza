# Cashenza custom-bundle Screenshot Brief

## Goal
Show that Cashenza custom-bundle is clean, easy to configure, and built to improve conversion on product pages.

## Screenshot 1
Title: Build bundle offers fast
Caption: Create 1-pack, 2-pack, 3-pack, or fully custom bundle ladders from one clean admin flow.

What to show:
- Bundle configurator open
- Clear item structure
- Multiple offers visible
- Merchant-friendly editing layout

## Screenshot 2
Title: Let customers choose variants per item
Caption: Show variant selectors inside the selected offer so customers can build the bundle they want.

What to show:
- Storefront bundle block
- Two or three item offer selected
- Variant selectors visible only for active offer
- Product name and variant name clearly displayed

## Screenshot 3
Title: Highlight the best seller offer
Caption: Push the recommended bundle with a best seller badge, savings badge, and stronger visual hierarchy.

What to show:
- Best seller badge aligned cleanly
- Save badge visible
- A premium-looking selected offer card

## Screenshot 4
Title: Match your brand style
Caption: Customize colors, spacing, radius, badges, and design presets directly in Shopify.

What to show:
- Theme editor block settings
- Preset selector
- Primary color and style settings
- Timer and badge styling options

## Screenshot 5
Title: Apply real Shopify-native discounts
Caption: Bundle discounts sync into Shopify so the customer sees the correct discounted cart result.

What to show:
- Bundle selected on storefront
- Cart with discount applied
- Clear before and after pricing

## Design guidance
- Use one consistent product family for screenshots
- Favor light, premium storefront visuals
- Keep each screenshot focused on one promise
- Avoid clutter and tiny unreadable controls
- Add short, high-contrast captions only

## Suggested visual sequence
1. Admin creation
2. Storefront variant experience
3. Merchandising badges
4. Theme customization
5. Discount applied in cart
